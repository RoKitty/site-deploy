motivation

the almost 20 year old Roblox website is lacking, and the new desktop app isn't filling the void.

developers have been building extensions that improve the website for more than a decade. extensions like BTRoblox and Roblox+ are great, but what if we went even further?

intro

hi! i'm jmkdev. i saw how the Roblox experience on desktop was lacking and decided i could make something better.

since September 2022, i've been building an app called RoKitty.

RoKitty is a web app that provides a completely reimagined the Roblox experience, rebuilt from the ground up. it's not a theme or patch, but instead a completely new web app packaged in the form of a browser extension.

right now RoKitty isn't publicly available - this means you can't download it yet - but i'm making progress here and there for an eventual release, so make sure to follow along!! :3
